# Parameter Display Component

## Type

Editor helper component

## Purpose

Renders formatted parameter values, units, bool choices, and bit grids used by signal-like widgets.

## Typical Use Cases

- Shared signal presentation
- Bool input choices
- Bit selection UI

## Key Configuration

- Presentation model
- Value and unit font sizing
- Bit columns and choice layout

## Runtime Notes

This component is shared by item and signal widgets and centralizes typed parameter presentation.
