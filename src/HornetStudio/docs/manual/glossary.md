# Glossary

## Item

A host-managed data object that becomes available for visualization or updates through the registry path.

## Host Registry

The host-side registry path through which items become visible and updateable for UI-facing systems.

## UI Registry

A UI-facing view or lookup path that should observe or query host-registered state instead of inventing a separate source of truth.

## Function Registry

An independent registry for callable functionality that can operate on items without being structurally owned by them.

## Widget

A configurable UI building block that visualizes, edits, or interacts with host-managed state.

## Producer

An internal or external source that creates or updates items before they are submitted to the host registry.
