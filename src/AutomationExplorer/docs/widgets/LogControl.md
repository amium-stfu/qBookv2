# LogControl Widget

## Type

`LogControl`

## Purpose

Displays log output from a configured log target inside the page.

## Typical Use Cases

- Host log monitoring
- Process output display
- Runtime diagnostics on a dashboard page

## Key Configuration

- Target log path
- Caption and footer
- Theme-aware layout settings

## Runtime Notes

The widget can be used for process and host-oriented log visualization inside normal pages and table cells.
