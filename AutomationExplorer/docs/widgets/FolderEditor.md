# FolderEditor Component

## Type

Editor surface component

## Purpose

Provides the visual page editor used to place, select, and add widgets.

## Typical Use Cases

- Add widgets to a page
- Move and resize widgets
- Manage table and list insertion workflows

## Key Configuration

- Current folder binding
- Grid line styling
- Selection and add-mode workflows

## Runtime Notes

The editor exposes add actions for the persisted control kinds and is a useful reference point for the supported widget set.
