# Widget Documentation Index

This folder contains one Markdown document per widget type.

## Folder Contract

- Location: `AutomationExplorer/docs/widgets/`
- One file per widget type
- File name matches the persisted widget `Type` value
- Example lookup: `AutomationExplorer/docs/widgets/<Type>.md`
- Detailed help file lookup: `AutomationExplorer/docs/widgets/help/<Type>.help.md`
