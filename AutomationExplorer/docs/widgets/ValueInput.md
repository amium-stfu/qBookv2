# ValueInput Editor Component

## Type

Editor helper component

## Purpose

Provides the in-app value editing surface for writable targets.

## Typical Use Cases

- Edit text values
- Edit numeric values
- Edit hex values
- Toggle bit-based values

## Key Configuration

- Editor mode based on target parameter kind
- Theme-aware input and button colors
- On-screen input pads

## Runtime Notes

This is an editor support component rather than a persisted top-level widget type, but it is relevant for widget-related in-app documentation.
