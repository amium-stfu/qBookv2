# TableControl Widget

## Type

`TableControl`

## Purpose

Provides a grid-based container for placing child widgets into rows and columns.

## Typical Use Cases

- Dashboard compositions
- Structured widget placement
- Mixed layouts with signals, charts, logs, and clients

## Key Configuration

- Row and column count
- Child cell position and span
- Container styling and theme properties

## Runtime Notes

The table manages child placement, cell visibility rules, and proportional child sizing based on cell span.
