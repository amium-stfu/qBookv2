# IMPLEMENTATION HANDOFF

## Goal
Rename the complete solution from `HornetStudio` / `HornetStudio` / `HornetStudio` naming to `HornetStudio` consistently across solution structure, projects, namespaces, imports, configuration, build files, hard-coded strings, and documentation references.

## Scope
Included:
- Rename solution file and solution launch file from `HornetStudio.*` to `HornetStudio.*`.
- Rename project folders and project files that contain `HornetStudio` or `HornetStudio` branding.
- Rename namespaces, `using` statements, Avalonia `x:Class`, XAML `using:` mappings, root namespaces, assembly names, project references, and asset URIs.
- Update config files, launch settings, CI/build files, documentation references, plugin output paths, and hard-coded user-facing strings that refer to the old product name.
- Run consistency checks and build validation.

Not included:
- No behavioral feature changes.
- No unrelated refactoring.
- No new runtime dependencies.
- No release versioning unless the user explicitly requests a release.
